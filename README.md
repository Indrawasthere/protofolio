# Muhammad Fadlan - Personal Blog

A minimalist, high-performance personal blog built with Next.js 15, Tailwind CSS, and TypeScript.

## Features
- **Clean UI**: Distraction-free reading experience inspired by modern design standards.
- **Responsive**: Perfectly scaled for mobile, tablet, and desktop.
- **Fast**: Static optimization and modern Next.js architecture.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide Icons

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
